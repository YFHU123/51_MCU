# 51_MCU 

本项目是一个基于 **8051 系列单片机** 的嵌入式开发学习仓库。包含了从底层 GPIO 控制到复杂传感器协议（1-Wire, I2C, SPI）的完整实现，旨在记录开发过程并提供可复用的模块化驱动。

## 🚀 项目亮点

* **模块化设计**：驱动代码提取至 `Module` 文件夹，方便快速移植到新项目。
* **开发环境友好**：完美适配 **VS Code + EIDE** 开发流程。
* **仿真支持**：提供 Proteus 仿真电路，无需硬件即可验证逻辑。
* **全中文注释**：关键逻辑均附带中文说明，易于理解。

## 📂 目录结构

| 目录 | 说明 |
| :--- | :--- |
| **Docs** | 硬件原理图、芯片 Datasheet 及相关技术文档 |
| **Examples** | 各类实验案例（流水灯、数码管、LCD、传感器等） |
| **Module** | 封装好的通用驱动库（如 1-Wire, I2C, ds1302 等） |
| **simulation** | Proteus 仿真工程文件 |

## 🛠️ 实验案例清单

| 编号 | 实验项目 | 核心技术点 |
| :--- | :--- | :--- |
| 01-04 | 基础 IO 交互 | 流水灯控制、数码管静态/动态显示、独立/矩阵按键扫描 |
| 05 | 串口通信 (UART) | 串口收发逻辑、波特率发生器、SBUF 处理 |
| 06-07 | LCD1602 显示 | 1602 驱动时序、自定义字符、电子时钟逻辑 |
| 08 | 8x8 点阵屏 | 74HC595 级联控制、图像取模显示 |
| 09-10 | DS18B20 温度计 | **1-Wire 单总线协议**、循环冗余校验、温度线性转换 |
| 11-12 | DS1307 时钟 | **I2C 总线协议**、RTC 寄存器读写、BCD 码处理 |
| 13 | A/D 模数转换 | 模拟信号采集、电位器数值读取 |
| 14 | 红外接收 | **NEC 协议**解码、外部中断触发、时序捕获 |

## 💻 开发环境

* **编译器**: Keil C51 
* **编辑器**: VS Code (推荐插件: `EIDE`)
* **烧录工具**: STC-ISP
* **仿真**: Proteus 8.15+
